number of feature choose	Output: [X, Y] as data which has k features chosen randomlyRandomForest(N, k, data): random forest algorithm	Input: N Ð tree number, k Ð feature number, data Ð raw data [X, Y]	Output: testset as [X, Y] and prediction for test data setgetPredict(combine):  vote for decision trees	Input: each prediction result of weak decision	Output: result of voteClass RF(MRJob): map reduce for random forestHow to run on aspen:Install necessary dependencies on aspenCopy input and RFMapReduce.py filesRun map reduce for random forest on aspenCommand lines:scp /Users/yijia/PycharmProjects/DecisionTree/RFMapReduce.py yijiaj@aspen.lti.cs.cmu.edu:/home/yijiajscp /Users/yijia/Desktop/input/* yijiaj@aspen.lti.cs.cmu.edu:/home/yijiaj/inputscp -r /Users/yijia/PycharmProjects/DecisionTree/input/* yijiaj@aspen.lti.cs.cmu.edu:/home/yijiaj/input/.[yijiaj@aspen ~]$ hdfs dfs -put /home/yijiaj/input /user/yijiaj/hadoop/[yijiaj@aspen ~]$ hdfs dfs -ls user/yijiaj/hadoop/inputscp /Users/yijia/Downloads/Anaconda2-4.3.1-Linux-x86_64.sh yijiaj@aspen.lti.cs.cmu.edu:/home/yijiajbash Anaconda2-4.3.0-MacOSX-x86_64.shexport PATH=~/anaconda2/bin:$PATHconda install -c conda-forge mrjob=0.5.5hdfs dfs -put /home/yijiaj/RFdata.csv /user/yijiaj/RFdata.csv[yijiaj@aspen ~]$ python RFMapReduce.py /home/yijiaj/input
